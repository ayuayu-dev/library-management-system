# 図書館管理システム

Laravel 12・Vue 3・Laravel Breezeを使用して開発した図書館管理システムです。

## 概要

Laravelの学習を目的として制作しました。

図書館での書籍管理や貸出・返却業務を想定し、実際の運用を意識したデータベース設計・画面構成を行っています。

## 使用技術

- PHP
- Laravel 12
- Laravel Breeze
- Vue 3
- MySQL


## 主な機能

- ログイン認証
- 書籍情報の登録・編集・検索
- 蔵書（冊子）管理
- 貸出・返却機能
- ページネーション
- 検索機能
